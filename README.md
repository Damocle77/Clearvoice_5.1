## üéõÔ∏è Preset Disponibili

<details>
<summary>üé¨ <strong>--film</strong> - Cinema/Action</summary>

**Ottimizzato per contenuti cinematografici con action e dialoghi intensi**

| Parametro | Valore | Descrizione |
|-----------|--------|-------------|
| **VOICE_VOL** | 8.5 | Boost dialoghi bilanciato |
| **LFE_VOL** | 0.24 (rid. ~17-20%) | Controllo sub per SP7 (varia per codec) |
| **SURROUND_VOL** | 3.6 | Effetti ambientali (varia per codec) |

- **LFE:** Crossover 25-105Hz (Dolby) / 30-115Hz (DTS) con precisione
- **Limitatore:** Cinematografico con preservazione dinamica

</details>

<details>
<summary>üì∫ <strong>--serie</strong> - Serie TV/Dialoghi</summary>

## üîç Troubleshooting

<details>
<summary>‚ùå Errori Comuni</summary>

### "File non 5.1 compatibile"
Lo script processa solo file con traccia audio 5.1. La validazione v0.78 fornisce suggerimenti automatici:
```bash
# Lo script rileva automaticamente il formato e suggerisce:
# MONO rilevato
# üí° Conversione: ffmpeg -i "file.mkv" -af "pan=5.1|FL=FC|FR=FC|FC=FC|LFE=0|BL=0|BR=0" -c:v copy output_51.mkv

# STEREO rilevato  
# üí° Upmix a 5.1: ffmpeg -i "file.mkv" -af "surround" -c:v copy output_51.mkv

# 7.1 SURROUND rilevato
# üí° Downmix a 5.1: ffmpeg -i "file.mkv" -af "pan=5.1|FL=0.5*FL+0.707*FLC|FR=0.5*FR+0.707*FRC|FC=FC|LFE=LFE|BL=BL|BR=BR" -c:v copy output_51.mkv
```

### "DTS encoder not supported"
```bash
# Verifica supporto DTS
ffmpeg -encoders | grep -i dts

# Soluzione: Usa EAC3 o AC3
./clearvoice078_preset.sh --film eac3 384k file.mkv
```

### Processing Parallelo Non Attivo
Il processing parallelo si attiva automaticamente solo per preset `--serie` con pi√π file:
```bash
# ‚úÖ Attiva processing parallelo (2 file contemporaneamente)
./clearvoice078_preset.sh --serie *.mkv          # Pi√π file
./clearvoice078_preset.sh --serie /series/dir/   # Cartella con pi√π file

# ‚ùå Processing sequenziale (altri preset o singolo file)
./clearvoice078_preset.sh --film file.mkv        # Singolo file
./clearvoice078_preset.sh --cartoni *.mkv        # Altri preset
```
</details>
